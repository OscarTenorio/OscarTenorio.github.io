[ Real Time Bus Tracker ]

Here we are tapping into the publicly available bus route data from Boston's MBTA, and displaying live feed of one of their current bus positions.

This is an exercise for applying asynchronous functions in managing promises to display live data - in this case the location of a specific bus (id: y1923). The asynchronous run() function is called every 15 seconds to not pass my request limit with Mapbox, so you'll have to wait a little in between updates. The bus route chosen is quite long, so in the event that you don't see the bus location marker, feel free to scroll down on the map to zoom out to see the bus location come into focus. I started the map zoom setting quite far for this reason, but mentioning it just in case.

Future improvements will see the ability to accept user input to search for different bus routes, and add functionality to change color schemes of the map display (everyone loves dark mode, eh?)
