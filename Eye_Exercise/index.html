[ Eyes Exercise ]

This exercise was simple - we were to essentially learn how elements on a page can react to the cursor using css animations. The eyes themselves are really just circle divs nested in another element that transform their position based on the cursor movement. So cute, but so creepy!

Unfortunately there are problems on the page - make the screen too narrow and the eyes stack on top of eachother. Further improvements are aimed at rectifying that, but for now the watcher prefers his page width to be above 770px.

Behold the watcher! Move the cursor around to watch them be a peepin' creeper!
