[ Eyes Exercise ]

An exercise demonstrating responsive CSS animation-like effects

>

This exercise was simple - we were to essentially learn how elements on a page can react to the cursor exclusively using CSS. The eyes themselves are really just circle divs nested in another element that transform their position based on the cursor movement. So cute, but so creepy! 👀

Unfortunately there are problems on the page - make the screen too narrow and the eyes stack on top of eachother. Further improvements are aimed at rectifying that, but for now the watcher prefers his page width to be above 770px.

You'll notice the eyes don't quite reflect where the cursor is, rather where it is in relation to the viewport. Future improvements will see to trying to get the eyes to follow where the cursor is in relation to the eyes themselves. More to come!
